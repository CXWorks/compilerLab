## 环境依赖
* Python 2.7(没有在3.5的情况下测试)
* networkx >= 1.11
* matplotlib >= 1.52
* Cheetah >= 2.4.4

其中networkx是Python的图论包，用于构建nfa与dfa。
</br>Cheetah是模板引擎，用于生成自动化代码
</br>然后matplotlib可以图形化图，更加直观
</br>麻烦助教安装一下了

为了完成一个近似Lex的pyLex！！！
