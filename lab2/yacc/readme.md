## 使用说明

### 用例文法

这个文法将贯穿整个实验报告作为例子，在[rules.y](rules.y)中:

    s -> s + t
    s -> t
    t -> t * f
    t -> f
    f -> ( s )
    f -> i

### .y文件要求

由于实现的是yacc的部分，因此只需要写出目标文法即可。我是采用空格进行分词的，因此所有符号间都要有空格
具体请见实例文件
